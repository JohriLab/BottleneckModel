# BottleneckModel
Bottleneck_Model_0.4.slim is the bottleneck model with no deme drift

bottleneck_simulation.py is a python script that simulates a single replicate of the bottleneck model

analysis_single_point.R calculates π and FST for a single point in the parameter space. It also plots the SFS for the bottleneck model against theoretical calculations for a simple WF model SFS.

The plotting folder contains the code for plotting the theoretical π and FST for all four models, as well as code for plotting the bottleneck model with simulated data for k = 10 and m = 0.4.
